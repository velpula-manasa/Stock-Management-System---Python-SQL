# Stock-Management-System---Python-SQL
Developed a Stock Management System using Python and SQLite to manage product details, categories, quantity in stock, unit price, and minimum stock levels. The system allows users to add, update, delete, and view products, helping track stock availability and maintain efficient product management.

The Stock Management System is a Python-based application developed to manage and monitor product stock efficiently. It allows users to store and manage product details such as product name, category, quantity in stock, unit price, and minimum stock level. This system helps in organizing stock records and ensuring proper product tracking.

## Features
- Add new products
- Update existing product details
- Delete products from the system
- View all available stock records
- Track quantity and unit price
- Maintain minimum stock level

## Technologies Used
- Python
- SQLite
